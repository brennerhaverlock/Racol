ðŸ“‹ Racol is a color module in node that's allows the user to create new colorscapes that fit well with any starting website, or even games needed for random color generation

## Features

* Get a random color value in HEX format
* Create color sets for MVP's on the fly
* Based on coolors.co's color generation


## Installation

Install globally 

```bash
npm install -g racol
```

## Usage

```bash
  Usage
  Locally
  $ colors = require('./index'); 
  Package
  $ colors = require('Racol')


  Examples
    $ colors.getRandomColor();
    $ colors.getRandomColorSet():
    $ colors.getAllColors()
  
```
